一、帖子标准

  • 为公开可见的小红书帖子。
  • 正文不少于 50 个汉字。
  • 至少包含 2 张 PaperChat 真实使用截图。
  • 正文中同时出现 PaperChat 和 Zotero。
帖子必须是你真实使用后的分享，不接受纯口号式推广、无实际体验描述或明显灌水内容。

二、奖励规则

基础奖励

发一篇符合标准的活动帖子，通过审核后即可获得：

  • 50w token

首期活动加码

前 20 位通过审核用户，额外获得：

  • +50w token
点赞数 额外奖励
达到 20 赞 +20w token
达到 50 赞 +50w token
达到 100 赞 +100w token
此后每增加 100 赞 +50w token

点赞追加奖励按同一篇活动帖子累计计算；达到对应门槛后，可再次联系客服申请。

三、不计入奖励的内容

  • 非公开可见帖子。
  • 搬运、洗稿、抄袭内容。
  • 纯广告口号、无实际使用描述的水帖。
  • 未展示真实产品使用截图的帖子。
  • 违规内容、虚假数据、恶意刷赞行为。
  • 与 PaperChat 无关的内容。

四、参与限制

  • 每位用户可提交 多 篇帖子参与审核。但不可重复领取已获得的奖励。

五、领奖流程

  1. 按活动要求，在小红书发布公开帖子。
  2. 点击当前页面上方“联系客服领奖”，扫码或打开淘宝链接联系PaperChat店铺客服。
  3. 将帖子链接和个人主页及帖子截图发给客服，等待审核并发放奖励。
